# Sudoku Solver

## Description

The application uses backtracking and recurssion to solve the famous Sudoku number game.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)

## Installation

Run the following code inside each part's folder

```
make
```

## Usage

Run the program by

```
./main
```

To clean up the object files, run

```
make clean
```
